### Stack
- Gatsby 
- Headless CMS (DatoCMS)
- TailwindCSS
- Emotion JS-in-CSS
- react-i18next
- eslint / prettier

### Installation
- `git clone https://github.com/newtelco/newtelco-dato`
- `cd newtelco-dato && npm install`
- `cp .env.example .env`
- Update `.env` variables
- `npm run dev`
