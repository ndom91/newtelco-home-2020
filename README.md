- Gatsby 
- DatoCMS
- TailwindCSS
- Emotion
- Framer Motion
- react-i18next
- eslint / prettier


## 🏗️ Installation
1. `git clone https://github.com/newtelco/newtelco-dato`
2. `cd newtelco-dato && npm install`
3. `cp .env.example .env`
4. Update `.env` variables
5. `npm run dev`
